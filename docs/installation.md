# OMGUI - Installation

> [!TIP]  
> _Recommended: create a virtual environment_
>
> ```shell
> python -m venv .venv
> ```
>
> ```shell
> source .venv/bin/activate
> ```

### Basic Install

```shell
pip install git+https://github.com/themoenen/omgui.git@v0.1
```

### Full Install

```shell
pip install git+https://github.com/themoenen/omgui.git@v0.1[viz]
```
